d.lirium
========

d.lirium is a personal blog software using vibe.d and the D programming language.

## Implemented Features

*   MongoDB backend
*   Creating articles
*   Editing articles
*   Writing comments
*   Deleting comments

## Planned Features

*   Redis for caching content
*   Deleting articles
*   Authentication via OAuth
    *   Google
    *   Github
*   Authorization and permissions
*   Short user profiles
*   Static pages
    *   About me
    *   ...
